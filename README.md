# 2026-National-Medical-Datathon-Investigation
This study used the Wisconsin Breast Cancer dataset, available at https://archive.ics.uci.edu/dataset/17/breast+cancer+wisconsin+diagnostic. This dataset consists of physical features of cell nuclei in breast tissue masses, along with labels of tumour status for each neuclei. This dataset has previously been used to investigate and evaluate performance of predictive models, mainly neural networks and support vector machines. This investigation expands such work to evaluate the performance of less computationally-expensive models, such as Random Forest and XGBoost.
Analysis was done in RStudio.
